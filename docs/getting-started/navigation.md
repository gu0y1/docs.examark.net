# 界面与导航

Examark 的主界面由侧边栏、顶栏和考试工作区组成。

## 侧边栏

侧边栏提供全局入口：

- 工作台。
- 考试管理。
- 班级管理。
- 学生管理。
- 管理员设置。

不同身份看到的数据不同。普通教师不应依赖前端隐藏作为权限边界，后端会继续按角色和能力裁剪字段。

## 顶栏

顶栏包含：

- 面包屑导航。
- 学校名称。
- 全局加载进度条。

面包屑示例：

```text
考试管理 > 2027-2028 学年 > 第一学期 > 七年级化学阶段测验
```

## 考试工作区

进入某场考试后，顶部显示考试工作区标签：

- 概览。
- 设计与印刷。
- 答题卡扫描。
- 阅卷。
- 成绩与报告。

系统会根据当前账号的 capability 裁剪入口。例如，扫描人默认只能看到扫描相关入口，阅卷人默认只能看到阅卷相关入口，试题编写人可以进入设计与印刷。
