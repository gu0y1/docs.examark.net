# 界面与导航

本页帮助你认识 Examark 的常用入口。不同账号看到的入口可能不同，这是权限裁剪的结果。

## 适用角色

- 所有学校用户。
- 第一次登录 Examark 的教师。

## 你将完成什么

- 认识左侧侧边栏。
- 认识考试工作区顶部标签。
- 知道为什么不同用户看到的入口不同。

## 开始前需要

- 已经通过组织入口登录，例如 `/org/demoSchool/grading`。
- 当前账号至少能进入工作台。

## 侧边栏

侧边栏提供全局入口：

- 工作台。
- 考试管理。
- 班级管理。
- 学生管理。
- 教师通讯录。
- 管理员设置。

管理员设置只会在当前账号拥有相关管理权限时显示。没有权限时，这个入口不会出现。

## 顶栏

顶栏通常包含：

- 当前页面位置。
- 当前学校或考试上下文。
- 当前登录用户入口。

点击右上角用户入口，可以进入账户信息或退出登录。具体可见项以当前版本界面为准。

## 考试工作区

进入某场考试后，顶部会出现考试工作区标签：

- 概览。
- 设计与印刷。
- 答题卡扫描。
- 阅卷。
- 成绩与报告。

每个标签下还有更细的模块。例如：

| 标签 | 常见模块 |
| --- | --- |
| 设计与印刷 | 试题配置、预印、印刷数据 |
| 答题卡扫描 | 扫描批次、录入结果、答题卡统计、出勤统计、交付阅卷 |
| 阅卷 | 阅卷分配、AI阅卷、客观复核、主观阅卷 |
| 成绩与报告 | 成绩、报告 |

## 为什么我的入口和别人不同

Examark 会根据你的学校角色、考试内角色和学生身份字段权限显示入口。

常见情况：

- 扫描人员通常能看到答题卡扫描，但不一定能看到成绩报告。
- 阅卷教师通常能看到阅卷，但默认看不到学生姓名。
- 试题编写人通常能看到设计与印刷，但不默认看到扫描原卷。
- 管理员或教务人员可能能看到管理员设置里的部分标签。

## 完成后你应该知道

- 日常考试从侧边栏的“考试管理”进入。
- 单场考试的操作集中在考试工作区。
- 看不到某个入口时，先确认自己是否有对应角色或权限。

## 常见问题

**为什么同一个老师在不同考试里看到的标签不同？**

考试内角色按单场考试配置。你在 A 考试里可能是阅卷人，在 B 考试里可能没有被加入协作人员。

**为什么管理员设置里有些标签看不到？**

管理员设置按具体权限显示。例如只有校历权限时，可能只看到校历设置；没有角色管理权限时不会看到角色管理。
