# 登录与组织入口

Examark 支持多组织使用。每个学校或组织都有自己的入口，用户需要先选择组织，再使用本组织账号登录。

## 适用角色

- 所有学校用户。
- 学校管理员。
- IT / 运维人员。

## 你将完成什么

- 了解推荐入口格式。
- 知道如何进入示例组织。
- 知道旧入口出现提示时应该怎么办。
- 理解组织入口和平台入口的区别。

## 开始前需要

- 你已经知道学校的组织简称，本文用 `demoSchool` 作为示例。
- 你的学校管理员已经为你创建教师账号。
- 登录用户名通常是教师工号。

## 推荐入口

租户业务入口使用：

```text
/org/{orgSlug}/grading
```

示例组织入口：

```text
/org/demoSchool/grading
```

进入后，如果尚未登录，系统会跳转到当前组织的登录页：

```text
/org/demoSchool/login
```

## 操作步骤

1. 打开 Examark 首页。
2. 搜索或选择你的学校。
3. 进入组织登录页。
4. 使用教师工号和密码登录。
5. 登录后进入本组织的工作台。

## 完成后你应该看到

- 页面进入当前组织的工作台。
- 左侧出现工作台、考试管理、班级管理、学生管理、教师通讯录等入口。
- 如果你有对应权限，还会看到管理员设置。

## 旧入口说明

旧入口 `/grading` 和 `/login` 不再作为推荐入口。正常教程不会继续使用旧入口。

如果你访问旧入口并看到提示，请根据页面提示切换到组织入口，例如：

```text
/org/demoSchool/grading
```

## 平台入口和组织入口

平台入口用于平台管理员管理组织目录、组织类型和平台设置；学校用户日常考试操作使用组织入口。

| 入口 | 面向谁 | 主要用途 |
| --- | --- | --- |
| `/platform` | 平台管理员 | 管理组织、平台设置和平台账号 |
| `/org/{orgSlug}/grading` | 学校用户 | 创建考试、扫描、阅卷、成绩发布 |

## 常见问题

**我不知道学校的 orgSlug 怎么办？**

打开首页，搜索学校名称或学校别名。找不到时联系学校管理员或 Examark 支持人员。

**为什么我登录后看不到管理员设置？**

管理员设置会按权限显示。只有具备学校基础信息、校历、年级、科目、角色、教师或学生相关管理权限的账号才会看到对应入口。

**我能用平台账号登录学校组织吗？**

不能。平台账号和学校教师账号是分开的。学校用户应使用本组织的教师账号登录。
