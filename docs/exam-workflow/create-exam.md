# 新建考试

## 入口

**考试管理 > 新建考试**

## 适用角色

- 管理员。
- 教务长。
- 具备考试维护权限的学校级角色。

## 操作步骤

1. 进入 **考试管理**。
2. 按需选择学年、学期或科目筛选。
3. 点击 **新建考试**。
4. 选择学年、学期、科目和年级。
5. 填写考试标题和说明。
6. 保存后进入考试详情。

## 字段说明

| 字段 | 说明 |
| --- | --- |
| 学年 | 考试所属教学年度 |
| 学期 | 考试所属时间段 |
| 科目 | 试题、任课教师和成绩归属所依赖的科目 |
| 年级 | 考试的年级范围 |
| 考试标题 | 页面、打印件和报告中展示的名称 |
| 说明 | 用于记录考试备注 |

## 历史年级快照

考试会保存年级快照。即使后续班级升年级，历史考试仍保留创建时的年级语义。

## 常见问题

**找不到学年或科目怎么办？**

到 **管理员设置** 先维护校历和科目。

**新建后没有学生怎么办？**

检查是否已经配置考核班级，并确认班级中存在 active 学生。
