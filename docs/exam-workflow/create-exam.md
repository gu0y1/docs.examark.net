# 新建考试

## 适用角色

- 管理员。
- 教务长。
- 具备考试维护权限的学校级角色。

## 你将完成什么

- 创建一场考试记录。
- 选择考试所属学年、学期、年级和科目。
- 进入考试工作区，开始配置班级和协作人员。

## 开始前需要

- 校历中已经有对应学年和学期。
- 年级和科目已经在管理员设置中创建。
- 当前账号具备创建考试的权限。

## 入口

**考试管理 > 新建考试**

## 操作步骤

1. 进入 **考试管理**。
2. 按需选择学年、学期或科目筛选。
3. 创建新考试。具体按钮名称以后续界面为准。
4. 选择学年、学期、科目和年级。
5. 填写考试标题和说明。
6. 保存后进入考试详情。

## 字段说明

| 字段 | 说明 |
| --- | --- |
| 学年 | 考试所属教学年度 |
| 学期 | 考试所属时间段 |
| 科目 | 试题、任课教师和成绩归属所依赖的科目 |
| 年级 | 考试的年级范围 |
| 考试标题 | 页面、打印件和报告中展示的名称 |
| 说明 | 用于记录考试备注 |

## 历史年级快照

考试会保存年级快照。即使后续班级升年级，历史考试仍保留创建时的年级语义。

## 完成后你应该看到

- 考试出现在考试管理列表中。
- 进入考试后可以看到考试概览。
- 下一步可以配置考核班级和协作人员。

## 常见问题

**找不到学年或科目怎么办？**

到 **管理员设置** 先维护校历和科目。

**新建后没有学生怎么办？**

检查是否已经配置考核班级，并确认班级中存在可用学生。

**新建考试按钮不可用怎么办？**

通常是当前账号没有创建考试权限，或当前筛选条件下缺少必需的学年、学期、年级、科目。先检查基础数据，再联系管理员确认权限。
