# 成绩发布与报告

## 适用角色

- 创建人。
- 考官。
- 超级管理员。
- 发布后具备成绩查看条件的相关教师。

## 你将完成什么

- 检查学生成绩汇总。
- 在确认无误后发布成绩。
- 按权限查看或导出成绩。
- 理解成绩发布和学生身份字段之间的区别。

## 开始前需要

- 客观复核和主观阅卷已经完成。
- 成绩已经重新计算或确认。
- 当前账号具备成绩查看或成绩发布权限。

## 入口

- **考试工作区 > 成绩与报告 > 成绩**
- **考试工作区 > 成绩与报告 > 报告**

## 成绩汇总

成绩记录包含：

- 客观题分数。
- 主观题分数。
- 调整分。
- 总分。
- 状态。

创建人、考官或超管可以重新计算成绩。

## 分数组成

| 分数类型 | 来源 | 常见处理人 |
| --- | --- | --- |
| 客观题分数 | 扫描识别和客观复核结果 | 扫描流程生成，阅卷教师复核异常 |
| 主观题分数 | 填空题和主观题阅卷结果 | 阅卷教师 |
| 调整分 | 特殊情况人工调整 | 考试创建人、考官或授权人员 |
| 总分 | 客观分、主观分和调整分汇总 | 系统计算，管理员确认 |

如果分数看起来不正确，先确认客观复核和主观阅卷是否完成，再重新计算成绩。

## 成绩发布

成绩初始为草稿状态。确认无误后发布成绩。发布会记录发布时间、发布人和发布状态。

## 发布前检查清单

- [ ] 所有扫描批次已经确认，不再有未处理异常卷。
- [ ] 客观题异常已经复核。
- [ ] 主观题和填空题阅卷任务已经完成。
- [ ] 成绩已重新计算。
- [ ] 抽查学生总分与题目分值一致。
- [ ] 导出字段符合当前账号权限预期。
- [ ] 已确认发布后哪些教师可以查看成绩。

## 发布不等于去匿名

成绩发布只改变成绩查看范围，不会自动授予学生身份字段。

能看成绩不代表能看到：

- 学生姓名。
- 学号。
- 班级。
- 扫描原卷。

## 导出

CSV 和 Excel 导出同样执行后端字段裁剪。匿名用户导出的第一列通常是匿名整卷 ID，而不是学生身份字段。

## 导出字段权限

| 当前账号权限 | 导出结果 |
| --- | --- |
| 只能看成绩 | 导出分数和匿名标识，不导出姓名、学号、班级 |
| 可看基础身份字段 | 可导出姓名、学号、班级等允许字段 |
| 可看敏感字段 | 仅在确有授权时导出敏感字段，普通成绩导出不建议包含 |
| 无成绩查看权限 | 不能导出成绩 |

导出不是前端截图，而是后端生成文件，因此即使页面上隐藏字段，导出也不会绕过权限。

## 完成后你应该看到

- 成绩状态为已发布。
- 有权限的教师可以查看成绩。
- 导出结果按当前账号权限显示字段。

## 常见问题

**发布成绩后为什么仍看不到学生姓名？**

成绩发布只开放成绩查看，不自动开放学生身份字段。姓名、学号、班级和扫描原卷仍按权限裁剪。

**为什么导出没有姓名和学号？**

导出使用同一套后端字段裁剪规则。没有学生身份权限时，导出会使用匿名 ID 或受限字段。

**总分没有变化怎么办？**

检查客观复核、主观阅卷和调整分是否已保存，再重新计算成绩。
