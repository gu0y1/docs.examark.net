# 预印与印刷数据

## 入口

- **设计与印刷 > 预印**
- **设计与印刷 > 印刷数据**

## 适用角色

- 创建人。
- 考官。
- 试题编写人。
- 超级管理员。

## 预印配置

预印阶段用于确认打印形态。

可配置：

- 试卷纸张：A4 或 A3 横向二合一。
- 答题卡纸张：A4 或 A3 横向二合一。
- 单面或双面。
- 空白页处理：留白或显示“此页留空”。
- 答题卡增强扫描双色打印。

## 预印预览

点击预览试卷和预览答题卡，检查版式、题号、答题区域和页码。

确认后点击 **预览通过**。预览通过后可进入印刷数据阶段。

## 生成印刷数据

在印刷数据页生成：

- 试卷 PDF。
- 答题卡整包 PDF。
- 每份答题卡的唯一条码。

答题卡份数可以手动输入，也可以使用默认 active 学生数。

## 重要限制

- 只有 active 印刷版本用于后续扫描识别。
- 已有扫描批次后，不应随意废弃当前 active 版本。
- 下载 PDF 需要 `can_print` 能力，无权限时后端不会返回文件 ID 或下载 URL。
