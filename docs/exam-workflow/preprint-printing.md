# 预印与印刷数据

## 适用角色

- 创建人。
- 考官。
- 试题编写人。
- 超级管理员。

## 你将完成什么

- 确认试卷和答题卡的打印形态。
- 生成用于实际印刷的 PDF。
- 确保后续扫描识别使用正确版本。

## 开始前需要

- 试题已经保存。
- 在线预览中试卷和答题卡版式已经检查。
- 当前账号能进入设计与印刷。

## 入口

- **考试工作区 > 设计与印刷 > 预印**
- **考试工作区 > 设计与印刷 > 印刷数据**

## 预印配置

预印阶段用于确认打印形态。

可检查或配置：

- 试卷纸张：A4 或 A3 横向二合一。
- 答题卡纸张：A4 或 A3 横向二合一。
- 单面或双面。
- 空白页处理：留白或显示“此页留空”。
- 答题卡增强扫描双色打印。

## 操作步骤

1. 进入预印。
2. 检查试卷和答题卡预览。
3. 确认纸张、单双面和空白页策略。
4. 预览无误后确认通过。
5. 进入印刷数据。
6. 生成试卷 PDF 和答题卡整包。
7. 下载并交给印刷人员。

## 重要限制

- 只有 active 印刷版本用于后续扫描识别。
- 已有扫描批次后，不应随意废弃当前 active 版本。
- 下载 PDF 需要打印权限；无权限时系统不会返回文件下载入口。

## 完成后你应该看到

- 试卷 PDF 可以下载。
- 答题卡 PDF 可以下载。
- 答题卡包含当前考试的条码、页码和答题区域。

## 常见问题

**为什么必须使用系统生成的答题卡？**

扫描识别依赖条码、页码和模板坐标。手工制作或修改答题卡会增加识别失败风险。

**打印后发现题目错误怎么办？**

先停止使用旧答题卡，回到试题配置修正内容，再重新完成预印和印刷数据。

**已经上传扫描件后还能重新印刷吗？**

通常不应这样做。已扫描的数据和 active 印刷版本强相关，强行更换可能导致识别和阅卷数据不一致。
