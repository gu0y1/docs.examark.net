# 配置考试协作

## 入口

**考试详情 > 概览 > 考试协作 > 配置协作**

## 适用角色

- 创建人 owner。
- 考官 examiner。
- 超级管理员。

## 考试内角色

| 角色 | 能做什么 | 默认不能做什么 |
| --- | --- | --- |
| 创建人 owner | 管理本场考试全部流程 | 不适用 |
| 考官 examiner | 管理大部分流程、参与人、题目分配和成绩发布 | 不转移 owner，不删除考试 |
| 试题编写人 paper_author | 试题设计、预印、印刷数据 | 不看身份、扫描、绑定、成绩 |
| 扫描人 scanner | 扫描上传、扫描原卷、答题卡绑定 | 不编辑试题、不阅卷、不发布成绩 |
| 阅卷人 grader | 客观复核、主观阅卷 | 默认匿名，不看学生身份 |

## 配置试题编写人

1. 在考试概览点击 **配置协作**。
2. 在 **试题编写人** 行选择教师。
3. 保存配置。
4. 切换到该教师身份时，应能进入 **设计与印刷**。

试题编写人可以保存试题、上传题图、在线预览、进入预印和生成印刷数据。

## 配置扫描人

扫描人用于扫描和绑定流程。扫描人默认可以查看：

- 学生姓名。
- 学号。
- 班级。
- 扫描原卷。
- 答题卡绑定关系。

扫描人默认不能查看敏感字段和成绩报告。

## 配置阅卷人

阅卷人需要配合题目分配使用。没有分配题目时，普通阅卷人不应看到其他题目的阅卷任务。

## 放权原则

owner 或 examiner 可以对指定参与人额外授予身份信息能力。敏感字段查看能力只能由 owner 或超管授予。

放权会写入审计日志，审计记录字段名、资源和动作，不记录身份证号、电话、地址、家长信息等敏感明文。
