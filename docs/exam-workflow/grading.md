# 客观复核与主观阅卷

## 入口

- **阅卷 > 客观复核**
- **阅卷 > 主观阅卷**

## 适用角色

- 阅卷人。
- 创建人。
- 考官。
- 超级管理员。

## 客观复核

客观复核用于处理自动识别结果中的异常状态：

- 空白。
- 多选。
- 低置信度。
- 识别失败。

教师可修正识别答案或人工分数，系统会重新计算最终得分。

## 主观阅卷

主观阅卷用于填空题和主观题。

阅卷界面展示：

- 题号。
- 匿名答案 ID。
- 识别文本或答题内容。
- 安全题目裁剪图。
- 评分标准。
- 分数输入。
- 评语输入。

## 题目分配

普通阅卷人只能保存被分配题目的结果。后端保存客观复核、主观评分、AI 评分或评语时都会强校验题目分配。

## 匿名阅卷

普通阅卷人和 AI 教师默认不接收：

- `student_id`。
- 学生姓名。
- 学号。
- 班级。
- 答题卡绑定信息。
- 整卷原图。

阅卷上下文使用 `anonymous_submission_id`、`anonymous_answer_id`、`grading_task_id` 或 `result_id` 定位。
