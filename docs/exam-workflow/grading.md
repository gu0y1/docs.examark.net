# 客观复核与主观阅卷

## 适用角色

- 阅卷人。
- 创建人。
- 考官。
- 超级管理员。

## 你将完成什么

- 处理客观题自动识别中的异常结果。
- 为填空题和主观题录入分数和评语。
- 理解匿名阅卷下能看什么、不能看什么。

## 开始前需要

- 扫描件已经处理并交付阅卷。
- 阅卷任务已经分配。
- 当前账号是本场考试阅卷人，或具备管理阅卷的权限。

## 入口

- **考试工作区 > 阅卷 > 客观复核**
- **考试工作区 > 阅卷 > 主观阅卷**

## 客观复核

客观复核用于处理自动识别结果中的异常状态：

- 空白。
- 多选。
- 低置信度。
- 识别失败。

教师可修正识别答案或人工分数，系统会重新计算最终得分。

## 主观阅卷

主观阅卷用于填空题和主观题。

阅卷界面通常展示：

- 题号。
- 匿名答案 ID。
- 识别文本或答题内容。
- 安全题目裁剪图。
- 评分标准。
- 分数输入。
- 评语输入。

## 题目分配

普通阅卷人只能保存被分配题目的结果。保存客观复核、主观评分、AI 评分或评语时，系统都会校验当前教师是否能处理该题。

!!! info "当前版本状态"
    当前版本以人工客观复核和人工主观阅卷为主。AI 辅助阅卷能力按学校配置和后续版本开放情况使用。

## 匿名阅卷

普通阅卷人和 AI 教师默认不接收：

- 学生姓名。
- 学号。
- 班级。
- 答题卡绑定信息。
- 整卷原图。

阅卷上下文使用匿名答案、阅卷任务或结果记录定位。

## 完成后你应该看到

- 客观题异常结果已被确认或修正。
- 主观题和填空题已经录入分数。
- 已保存的任务不再显示为空任务。
- 成绩页重新计算后能反映阅卷结果。

## 常见问题

**为什么我看不到学生姓名？**

这是匿名阅卷设计。普通阅卷人默认只看到匿名答案、题目内容和裁剪区域。

**为什么我的主观阅卷页面没有任务？**

请确认扫描已经交付阅卷，并且考试创建人或考官已经在阅卷分配中把题目分配给你。

**保存分数失败怎么办？**

检查当前题目是否分配给你、分数是否超过题目满分，以及当前考试是否仍允许阅卷。
