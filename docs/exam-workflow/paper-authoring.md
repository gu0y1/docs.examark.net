# 试题编写

## 入口

**考试工作区 > 设计与印刷 > 试题配置**

## 适用角色

- 创建人。
- 考官。
- 试题编写人。
- 超级管理员。

## 页面结构

试题配置页包含：

- 试卷标题、题目数量和总分。
- 试卷信息配置。
- 历史版本。
- 题组和题目栏。
- 当前题目编辑器。
- 实时预览和答题卡区域预览。

## 编写题目

1. 新增或选择题组。
2. 点击 **新增题目**。
3. 选择题型。
4. 填写分值。
5. 编辑题干。
6. 对客观题填写选项和答案。
7. 对填空题插入空位并填写参考答案。
8. 对主观题填写评分标准。
9. 保存试题。

## 支持题型

| 题型 | 自动评分 | 说明 |
| --- | --- | --- |
| 单选题 | 支持 | 答案通常为 A、B、C、D |
| 多选题 | 支持 | 必须完全匹配正确选项 |
| 判断题 | 支持 | 使用固定选项 |
| 填空题 | 不自动 OCR 判分 | 进入人工阅卷 |
| 主观题 | 不自动判分 | 进入人工阅卷 |

## 富文本能力

试题编辑器支持：

- 加粗、斜体、下划线。
- 图片上传。
- inline 公式。
- 填空题空位控件。

图片作为试题素材保存，下载和展示仍受考试权限保护。

## 在线预览

保存后点击 **在线预览**。系统会展示：

- 学生试卷预览。
- 答题卡预览。

确认无误后点击 **进入预印**。进入预印后当前版本会锁定，继续编辑需要先废弃预印或印刷版本。
