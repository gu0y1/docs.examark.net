# 试题编写

## 适用角色

- 创建人。
- 考官。
- 试题编写人。
- 超级管理员。

## 你将完成什么

- 维护题组和题目。
- 设置分值、答案和评分标准。
- 检查试卷和答题卡预览。
- 为后续预印和扫描识别准备稳定版式。

## 开始前需要

- 考试已经创建。
- 当前账号能进入 **设计与印刷 > 试题配置**。
- 已经确定本场考试的题型、分值和评分标准。

## 入口

**考试工作区 > 设计与印刷 > 试题配置**

## 页面结构

试题配置页包含：

- 试卷标题、题目数量和总分。
- 试卷信息配置。
- 历史版本。
- 题组和题目栏。
- 当前题目编辑器。
- 实时预览和答题卡区域预览。

## 操作步骤

1. 新增或选择题组。
2. 新增题目。
3. 选择题型。
4. 填写分值。
5. 编辑题干。
6. 对客观题填写选项和答案。
7. 对填空题插入空位并填写参考答案。
8. 对主观题填写评分标准。
9. 保存试题。
10. 在线预览试卷和答题卡。

## 支持题型

| 题型 | 自动评分 | 说明 |
| --- | --- | --- |
| 单选题 | 支持 | 答案通常为 A、B、C、D |
| 多选题 | 支持 | 必须完全匹配正确选项 |
| 判断题 | 支持 | 使用固定选项 |
| 填空题 | 不自动 OCR 判分 | 进入人工阅卷 |
| 主观题 | 不自动判分 | 进入人工阅卷 |

## 富文本能力

试题编辑器支持常见富文本能力，例如加粗、斜体、下划线、图片上传、公式和填空题空位控件。图片作为试题素材保存，下载和展示仍受考试权限保护。

## 完成后你应该看到

- 题目数量和总分符合考试要求。
- 每道客观题都有正确答案。
- 填空题和主观题有评分标准。
- 在线预览中的试卷和答题卡版式正常。

## 常见问题

**为什么题号不能手动输入？**

题号由排序生成，避免试卷、答题卡和阅卷任务不一致。

**进入预印后还能改题吗？**

可以继续编辑，但通常需要废弃当前预印或印刷版本并重新确认。若已有扫描批次，系统会限制破坏已有数据的操作。

**试题图片上传后谁能看？**

图片仍受考试权限保护。能编辑、打印或阅卷的角色会按业务需要看到对应素材。
