# 用户手册

## Examark：面向学校真实考试场景的智能阅卷与考试管理系统

Examark 是一套面向学校真实考试场景设计的智能阅卷与考试管理系统。

它的核心目标，是让每一次考试都更容易组织、更快速批改、更清晰分析。平台支持教师从创建考试、设计试卷、生成答题卡、上传扫描件、自动识别客观题、分配主观题阅卷任务，到生成学生成绩、试卷档案和数据分析报告的完整流程。Examark 不只是帮助学校批改一张试卷，而是将传统考试流程数字化、结构化、智能化，让考试成为可追踪、可分析、可改进的学习记录。

## 面向教师

Examark 可以减少考试准备和阅卷过程中的重复性工作。教师可以在平台内完成从出题、组卷、排版到答题卡生成的一站式流程，缩短印刷前的准备时间，把更多精力放在题目质量、教学设计和学生反馈上。

阅卷环节中，系统支持客观题自动识别与评分；对于主观题，Examark 引入人工智能进行辅助判题、初筛与参考建议，帮助教师更快完成批改。面对大型考试，平台支持阅卷任务拆分、教师分配、进度追踪和质量控制，让多人协同阅卷更加有序、高效。

## 面向学生和家长

Examark 提供的不只是一个分数，而是一份可以理解、可以追踪、可以改进的学习反馈。家长可以查看孩子的考试表现、错题记录、知识点薄弱项和阶段性学习曲线，了解孩子真正需要帮助的地方。

学生也可以在平台中查看错题归纳、题目讲解、考试讲评和个人成长记录，从一次考试中获得更清晰的学习方向。Examark 希望帮助学生从知道自己错了，走向知道为什么错、下一步如何改进。

## 面向学校

Examark 是考试评价流程数字化转型的基础设施。系统围绕学校长期使用场景设计，支持自定义年级体系、班级管理、学生档案、教师档案、家长信息、科目设置、校历设置、角色权限、班主任与任课教师数据范围，以及 AI 教师配置。

学校可以在统一的平台中管理考试、阅卷、成绩、学生成长记录和教学数据，减少纸质流程、人工汇总和多系统割裂带来的管理成本。

## 系统集成

Examark 重视与学校现有系统的兼容能力。平台原生面向 LMS 与学校信息系统集成场景设计，可支持与 Canvas、Blackboard 等学习管理系统对接，并可根据学校现有 OA、教务、学生信息系统和数据平台进行定制化集成。

对于不同学校的年级结构、考试流程、权限制度和数据规范，Examark 可以通过灵活的角色权限、组织配置和接口适配进行支持，降低学校引入新系统的迁移成本。

## 长期价值

Examark 不只是一款智能阅卷工具，也是一套面向学校教学评价与学习管理的综合平台。它连接试卷、答题卡、阅卷、成绩、错题、教师、学生、家长和学校管理流程，帮助学校沉淀高质量的教学数据，支持学生成长过程的长期追踪，并为未来的 AI 辅助教学、个性化学习分析和学校数据治理打下基础。

Examark 希望让每一次考试都有迹可循，让每一份答案都成为学生成长的记录，让学校的考试评价从低效的纸面流程，走向智能、清晰、可持续的数据化管理。

## 适用对象

- **学校管理员与教务人员**：维护学校基础数据、教师、学生、角色权限和考试流程。
- **考官与考试创建人**：配置考试、组织协作人员、发布成绩。
- **试题编写人**：维护试题、题组、试卷信息、预印和印刷数据。
- **扫描人员**：上传扫描件、查看扫描原卷和答题卡绑定信息。
- **阅卷教师**：完成客观复核、主观阅卷和评语录入。
- **AI 教师**：作为匿名阅卷和辅助分析能力的预留角色。

```mermaid
flowchart LR
  A[创建考试] --> B[配置班级与协作人员]
  B --> C[设计试卷]
  C --> D[生成答题卡与印刷数据]
  D --> E[扫描上传]
  E --> F[客观题识别]
  F --> G[主观题阅卷]
  G --> H[成绩发布]
  H --> I[报告分析]
```
