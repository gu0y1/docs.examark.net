# 扫描与识别 SOP

扫描与识别是纸质考试进入阅卷系统的关键环节。本页按现场操作顺序列出推荐做法和常见异常处理。

## 适用角色

- 扫描人员。
- 考试创建人。
- 考官。
- 学校教务人员。

## 真实任务场景

考试结束后，你需要把一个班或一个年级的答题卡上传到 Examark。系统会识别条码、页码、学号填涂和客观题填涂，并裁剪填空题和主观题区域。

## 开始前需要

- 使用的是当前考试生成的答题卡。
- 答题卡没有混入其他考试或旧版本。
- 扫描件条码、页码、学号填涂区和客观题区域清晰可见。
- 当前账号能进入 **考试工作区 > 答题卡扫描**。

## 推荐扫描设置

| 项目 | 建议 |
| --- | --- |
| 分辨率 | 300 DPI 起步 |
| 颜色 | 灰度或彩色均可，正式场景优先灰度或彩色 |
| 文件格式 | PDF 优先，也支持 JPG / PNG |
| 页面方向 | 尽量保持正向，不要旋转或倒置 |
| 页面完整性 | 不裁掉条码、页码、边角和学号填涂区 |
| 文件组织 | 按班级或考场分批上传，便于排查 |

手机拍照可用于临时排查，但正式考试建议使用扫描仪。

## 标准操作步骤

1. 按班级或考场整理答题卡。
2. 检查是否混入其他考试或旧版本答题卡。
3. 扫描并保存为 PDF、JPG 或 PNG。
4. 进入 **考试工作区 > 答题卡扫描 > 扫描批次**。
5. 上传扫描件。
6. 等待系统处理批次。
7. 查看录入结果。
8. 处理异常页和人工绑定。
9. 查看答题卡统计和出勤统计。
10. 确认无误后交付阅卷。

## 上传后系统会做什么

系统会依次处理：

- 创建扫描批次。
- 拆分 PDF 或读取图片。
- 做页面归一化。
- 识别条码和页码。
- 按答题卡归组。
- 识别学号填涂。
- 识别客观题填涂框。
- 裁剪填空题和主观题区域。
- 生成客观复核和主观阅卷所需数据。

## 故障矩阵

| 现象 | 可能原因 | 处理步骤 | 是否可继续交付阅卷 |
| --- | --- | --- | --- |
| 条码识别失败 | 条码模糊、遮挡、缺角，或答题卡来自其他考试 | 查看原始扫描页，确认条码和考试来源，必要时重新扫描 | 不建议，来源确认前不要交付 |
| 学号绑定失败 | 学号填涂不完整、学生不在考核班级、重复答题卡 | 在录入结果中人工绑定学生，确认班级范围 | 处理完成后可以 |
| 缺页 | 扫描仪漏扫、双面设置错误、学生缺交 | 对照纸质答题卡补扫或标记缺页 | 缺页未确认前不要交付 |
| 重复页 | 扫描仪重复进纸或重复上传 | 删除或隔离重复页，保留清晰版本 | 清理后可以 |
| 上传错考试 | 答题卡标题、条码或版式与当前考试不一致 | 停止处理，删除或隔离批次，上传到正确考试 | 不可以 |
| 客观题识别异常 | 填涂过浅、多选、擦除不干净、框外作答 | 进入客观复核，由教师人工确认答案或分数 | 可以交付到客观复核 |
| 主观题裁剪异常 | 版式不匹配、扫描变形、答题卡版本错误 | 检查 active 印刷版本和扫描质量，必要时重新扫描 | 裁剪不可信时不要交付 |
| 大量学生未绑定 | 考核班级配置错误或学号规则不匹配 | 先检查考试班级和学生档案，再人工处理 | 大量异常未处理前不要交付 |

## 常见异常处理

### 条码识别失败

可能原因：

- 条码打印太淡。
- 扫描件模糊或过暗。
- 条码被折痕、污渍或装订遮挡。
- 上传了其他考试或旧版本答题卡。

处理方法：

1. 查看原始扫描页。
2. 确认条码是否完整。
3. 确认答题卡是否来自当前考试。
4. 必要时重新扫描或重新上传。
5. 不确定来源时，不要交付阅卷。

### 学号绑定失败

可能原因：

- 学生未按要求填涂学号。
- 学号位数不完整。
- 学生不在本场考试的考核班级中。
- 同一学生出现重复答题卡。

处理方法：

1. 进入录入结果。
2. 查看系统识别到的学号或异常提示。
3. 人工选择正确学生。
4. 确认绑定后再继续交付。

### 客观题识别异常

可能原因：

- 填涂过浅。
- 橡皮擦除不干净。
- 多选题未按要求填涂。
- 学生在框外作答。

处理方法：

1. 让系统生成客观复核任务。
2. 阅卷教师在客观复核中确认答案或人工分数。
3. 对大量同类异常，检查扫描质量或答题卡打印质量。

### 缺页或页码异常

可能原因：

- 扫描仪漏扫。
- 学生缺交某页。
- 双面扫描设置错误。
- 答题卡被打乱或重复扫描。

处理方法：

1. 对照纸质答题卡检查页数。
2. 重新扫描缺失页。
3. 对重复页或错误页进行删除或隔离。
4. 确认每份答题卡页码完整后再交付阅卷。

### 上传错考试

表现：

- 大量条码识别失败。
- 答题卡标题或版式与当前考试不一致。
- 识别结果无法正常归组。

处理方法：

1. 立即停止交付阅卷。
2. 通知考试创建人。
3. 删除或隔离错误批次。
4. 找到正确考试后重新上传。

## 交付阅卷前检查

交付阅卷前，至少确认：

- [ ] 扫描批次处理完成。
- [ ] 答题卡统计数量与实际收卷数量基本一致。
- [ ] 学号绑定失败和重复绑定已处理。
- [ ] 缺页、错页或作废页已经确认。
- [ ] 没有混入其他考试的答题卡。
- [ ] 客观题识别异常可以进入客观复核。
- [ ] 填空题和主观题裁剪区域已经生成。

## 完成后你应该看到

- 扫描批次和录入结果可以正常查看。
- 可交付的答题卡已经进入阅卷候选。
- 阅卷分配页面可以继续分配主观题任务。

## 常见问题

**扫描件倾斜还能识别吗？**

轻微倾斜通常可以处理。严重倾斜、透视变形、裁切边角或条码缺失会显著增加失败率。

**支持手机拍照吗？**

可以用于临时排查，但不建议作为正式考试的主要上传方式。正式场景应使用扫描仪。

**彩色扫描会影响识别吗？**

通常不会。关键是条码、页码和填涂区域清晰，且文件没有过度压缩。

**为什么扫描人能看到姓名，但阅卷人看不到？**

扫描人员需要处理学生绑定，因此可以看到必要身份信息。阅卷教师默认匿名，只需要看到题目答案和评分标准。
