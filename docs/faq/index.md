# 普通用户 FAQ

本页面向学校日常用户：教师、扫描人员、阅卷教师、考试创建人和教务人员。开发环境、部署和服务器问题请查看 [技术与部署](../technical/index.md)。

## 适用角色

- 学校用户。
- 第一次使用 Examark 的教师。
- 负责考试流程的教务人员。

## 登录与入口

**我应该从哪里登录？**

使用学校组织入口登录，例如 `/org/demoSchool/grading`。如果不知道组织入口，请在首页搜索学校名称或联系管理员。

**登录用户名是什么？**

通常使用教师工号。具体账号由学校管理员创建和维护。

**为什么我登录后看不到管理员设置？**

管理员设置按权限显示。没有学校基础信息、校历、年级、科目、角色、教师或学生相关管理权限时，不会显示该入口。

## 考试创建

**为什么我看不到某场考试？**

可能原因包括：

- 筛选条件选错学年、学期或科目。
- 你没有被加入该考试。
- 你没有查看该考试的权限。

先清空筛选或切换学期查看，再联系考试创建人确认权限。

**新建考试时没有学年、年级或科目可选怎么办？**

请管理员先在管理员设置中维护校历、年级和科目。

## 试题与答题卡

**进入预印后还能修改题目吗？**

可以继续编辑，但通常需要废弃当前预印或印刷版本并重新确认。已有扫描批次时，不应随意更换印刷版本。

**为什么必须使用系统生成的答题卡？**

扫描识别依赖答题卡条码、页码和模板坐标。手工制作或修改答题卡会增加识别失败风险。

## 扫描识别

**扫描失败怎么办？**

先确认：

- 答题卡来自当前考试。
- 条码清晰。
- 页码完整。
- 扫描件没有明显倾斜、模糊、缺角或过度压缩。

更完整步骤见 [故障处理](troubleshooting.md) 和 [扫描与识别 SOP](../scanning-printing/sop.md)。

**学号识别失败怎么办？**

进入录入结果，人工绑定正确学生。人工绑定后，系统会同步答题卡关系、识别摘要和阅卷任务关联。

## 阅卷评分

**为什么我没有阅卷任务？**

常见原因：

- 扫描结果还没有交付阅卷。
- 题目还没有分配给你。
- 你没有被加入本场考试协作人员。

请联系考试创建人或考官检查扫描交付和阅卷分配。

**为什么阅卷人看不到学生姓名？**

这是匿名阅卷设计。普通阅卷教师默认只看到匿名答案、题目裁剪图、识别文本和评分标准。

## 成绩导出

**成绩发布后为什么仍看不到学生身份？**

成绩发布只开放成绩查看，不自动开放学生姓名、学号、班级或扫描原卷。

**导出为什么没有姓名和学号？**

导出同样遵守字段裁剪。当前账号没有学生身份字段权限时，导出不会包含姓名和学号。

## 仍然无法解决

请记录：

- 组织名称。
- 考试名称。
- 当前账号。
- 你所在页面。
- 你正在执行的操作。
- 页面提示或错误信息。

然后联系学校管理员或 Examark 支持人员。
