# 考试创建人手册

考试创建人负责把一场考试从“考试记录”推进到“成绩发布”。你不一定亲自完成每个环节，但需要确认每个环节有人负责、状态可追踪、结果可交付。

## 适用角色

- 考试创建人。
- 考官。
- 教务人员。
- 负责组织年级考试的老师。

## 真实任务场景

你要组织 `2026 春季 七年级数学期中考试`，需要选择考试班级、安排试题编写人、安排扫描人员和阅卷教师，并在阅卷完成后发布成绩。

## 开始前需要

- 学年、学期、年级、科目、班级和学生已经准备好。
- 本场考试的试题编写、扫描和阅卷负责人已经确定。
- 你拥有创建考试和管理本场考试的权限。

## 每天怎么操作

1. 进入 **考试管理**，查看当前负责的考试。
2. 进入考试工作区的 **概览**，检查考试基本信息和协作人员。
3. 在 **设计与印刷** 中确认试题和印刷数据状态。
4. 在 **答题卡扫描** 中检查扫描批次、录入结果和交付状态。
5. 在 **阅卷** 中检查阅卷分配和阅卷进度。
6. 在 **成绩与报告** 中检查成绩是否可发布。

## 关键检查点

| 阶段 | 你要确认什么 |
| --- | --- |
| 新建考试 | 学年、学期、年级、科目和标题正确 |
| 配置协作 | 试题编写人、扫描人、阅卷人已配置 |
| 试题编写 | 试卷总分、题号、答案和评分标准完整 |
| 预印印刷 | active 印刷版本已生成，答题卡条码清晰 |
| 扫描上传 | 异常卷已处理，错误考试答题卡未交付 |
| 阅卷 | 题目已分配，阅卷任务不为空 |
| 成绩发布 | 客观分、主观分和调整分已确认 |

## 完成后你应该看到

- 考试进入可发布成绩状态。
- 阅卷任务完成，成绩表无明显异常。
- 发布后，相关教师按权限看到成绩。

## 出错怎么办

**考试列表里找不到这场考试**

检查筛选条件是否选错学年、学期或科目；也可能是当前账号没有查看这场考试的权限。

**扫描完成但阅卷人没有任务**

检查是否已在答题卡扫描中交付阅卷，并在阅卷分配中把题目分配给对应教师。

**成绩发布前发现题目或分值错误**

先暂停发布。确认是否需要回到试题配置重新修正，并评估是否需要重新生成印刷数据或重新扫描。

**发布后有人看不到学生姓名**

这是权限设计。成绩发布不自动开放学生身份字段。请阅读 [常见权限场景](../permissions/scenarios.md)。
