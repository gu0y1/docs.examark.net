# 扫描人员手册

扫描人员负责把纸质答题卡转换成系统中的扫描批次，并处理识别异常和学生绑定问题。

## 适用角色

- 扫描人员。
- 考试创建人。
- 考官。

## 真实任务场景

考试结束后，你收到 `七年级 1 班` 的答题卡，需要上传扫描件，检查识别结果，处理无法自动绑定学生的答题卡，并交付给阅卷流程。

## 开始前需要

- 本场考试已经生成当前 active 印刷版本。
- 学生实际使用的是本场考试生成的答题卡。
- 扫描件清晰，条码、页码、学号填涂区没有被裁掉。
- 你能进入 **考试工作区 > 答题卡扫描**。

## 每天怎么操作

1. 进入本场考试工作区。
2. 打开 **答题卡扫描 > 扫描批次**。
3. 上传扫描 PDF、JPG 或 PNG。
4. 等待系统处理批次。
5. 进入录入结果，查看条码、页码和学生绑定情况。
6. 处理学号识别失败、重复绑定、缺页和异常页。
7. 查看答题卡统计和出勤统计。
8. 确认无误后交付阅卷。

## 操作重点

- 不要把其他考试或旧印刷版本的答题卡上传到当前考试。
- 条码和页码比文件名更重要，文件名不能替代系统识别。
- 学号识别失败时，优先人工绑定正确学生。
- 异常卷未确认前，不要急于交付阅卷。

## 完成后你应该看到

- 扫描批次处理完成。
- 大部分答题卡已经绑定学生。
- 异常页有明确处理结果。
- 客观题识别结果、填空题裁剪和主观题裁剪已经生成。
- 交付阅卷后，阅卷分配可以继续处理。

## 出错怎么办

**条码识别失败**

检查扫描件是否模糊、过暗、缺角、压缩严重或条码被遮挡。必要时重新扫描。

**学号绑定失败**

进入录入结果，人工选择正确学生。确认学生是否在本场考试的考核班级中。

**缺页或多页归组异常**

检查答题卡页码是否完整，确认扫描仪是否漏扫、重扫或打乱顺序。

**上传错考试**

停止交付阅卷。联系考试创建人删除或隔离错误批次，并重新上传正确考试的答题卡。
