# 学生与家长档案

学生档案决定考试范围、成绩归属和报告查看。家长档案属于敏感信息，默认不会向普通教师开放。

## 适用角色

- 学校管理员。
- 教务人员。
- 班主任或具备学生管理权限的教师。

## 你将完成什么

- 维护学生基本信息。
- 把学生放入正确班级。
- 理解哪些字段属于敏感信息。
- 处理学生转班。

## 开始前需要

- 年级和班级已经创建。
- 当前账号具备学生查看或学生管理权限。
- 学校已经确认学生信息来源。

## 学生档案

学生档案通常包含：

- 姓名。
- 学号。
- 班级。
- 状态。
- 身份证号。
- 电话。
- 邮箱。
- 国籍。

其中身份证号、电话、邮箱、国籍等属于敏感字段，系统会按权限裁剪。

## 家长档案

家长信息包含联系方式、关系、主要联系人和紧急联系人等。家长信息按敏感字段规则裁剪。

!!! info "当前版本状态"
    当前版本支持学生与家长基础档案维护和权限裁剪。家长端报告和面向家长的批量推送能力以后续版本为准。

## 操作步骤

1. 进入 **学生管理**。
2. 按班级或姓名查找学生。
3. 新建或编辑学生档案。
4. 填写姓名、学号、班级和状态。
5. 如有权限，再维护敏感字段和家长信息。
6. 保存后确认学生出现在对应班级。

## 转班

学生可以从一个班级转入另一个班级。历史考试仍通过考试班级范围和当时学生记录保持历史语义。

转班前建议确认：

- 当前学生是否有正在进行中的考试。
- 转班是否影响后续考试班级范围。
- 是否需要通知班主任或任课教师。

## 完成后你应该看到

- 学生出现在正确班级。
- 新建考试选择班级后，能纳入对应学生。
- 没有敏感字段权限的教师不会看到身份证号、电话、家长信息等字段。

## 常见问题

**为什么我看不到学生电话或家长信息？**

这些属于敏感字段，需要额外权限。普通任课或阅卷流程通常不需要查看。

**学生转班后历史考试会变化吗？**

历史考试应保留当时考试范围，不会因为后续转班自动改变历史语义。

**导出学生信息时字段不完整怎么办？**

导出同样按权限裁剪。请联系管理员确认当前账号是否具备对应字段权限。
