# 教师、角色与绑定

教师、角色与绑定决定一位老师在学校范围内能看什么、能管理什么，以及在某些年级、科目或班级上的作用域。

## 适用角色

- 学校管理员。
- 教务人员。
- 具备教师管理、角色管理或教师绑定权限的教师。

## 你将完成什么

- 维护教师档案。
- 配置学校级角色。
- 把教师绑定到全校、年级、科目或班级。
- 理解学校级角色和考试内参与人的区别。

## 开始前需要

- 当前账号能进入管理员设置中的教师管理或角色管理。
- 已经准备好教师姓名、工号和联系方式。
- 已经明确教师的学校职责，例如班主任、任课教师、年级组长或教务人员。

## 教师管理

教师档案通常包含：

- 姓名。
- 工号。
- 类型：普通教师或 AI 教师。
- 电话和邮箱。
- 状态。

人工教师可以登录系统。AI 教师用于 AI 辅助阅卷等场景，默认不作为普通用户登录。

## 角色管理

角色由权限组成。学校可以根据管理制度创建角色，例如：

- 学校管理员。
- 教务长。
- 年级组长。
- 学科组长。
- 班主任。
- 任课教师。

!!! info "当前版本状态"
    当前版本支持学校级角色、教师绑定和考试内协作人员配置。AI 教师相关能力按学校配置和后续版本开放情况使用。

## 教师绑定

教师可以绑定到不同作用域：

| 绑定类型 | 示例 |
| --- | --- |
| 全校 | 教务长、系统管理员 |
| 年级 | 七年级组长 |
| 科目 | 数学学科组长 |
| 班级 | 七年级 1 班班主任 |
| 班级 + 科目 | 七年级 1 班数学任课教师 |

## 操作步骤

1. 进入 **管理员设置 > 教师管理**。
2. 新建或选择教师。
3. 填写姓名、工号、联系方式和状态。
4. 如需登录，设置或重置密码。
5. 在角色或绑定标签中添加学校级角色。
6. 保存后刷新教师列表，确认绑定数量和状态。

## 考试内参与人

考试内参与人不在教师管理页统一配置，而是在考试概览中按单场考试配置。

例如，同一位教师可以：

- 在学校中是普通任课教师。
- 在某场考试中是试题编写人。
- 在另一场考试中是阅卷教师。

## 完成后你应该看到

- 教师通讯录中能看到教师。
- 教师具备正确的学校级角色或作用域绑定。
- 教师登录后能看到对应入口。

## 常见问题

**教师有管理员设置入口，但缺少某个标签怎么办？**

说明该教师只有部分设置权限。补充对应权限后，刷新页面或重新登录。

**教师被设为阅卷人后为什么没任务？**

考试内阅卷人还需要题目分配。教师管理中的学校角色不会自动生成单场考试阅卷任务。

**可以给所有教师都开系统管理员吗？**

不建议。系统管理员权限过高，应只给少数负责全校系统配置的人。
