# 教师、角色与绑定

## 教师管理

教师档案包含：

- 姓名。
- 工号。
- 类型：普通教师或 AI 教师。
- 电话和邮箱。
- AI 模型配置。
- 状态。

AI 教师可以配置模型名称、接口地址、配置 JSON 和密钥引用。

## 角色管理

角色由权限复选框组成。角色可以属于：

- 全校。
- 年级。
- 科目。
- 班级。
- AI。

## 教师绑定

教师可以绑定到组织、年级、科目或班级。绑定决定学校级作用域。

示例：

- 班主任绑定到班级。
- 任课教师绑定到班级和科目。
- 年级组长绑定到年级。
- 学科组长绑定到科目。

## 考试内参与人

考试内参与人不在教师管理页统一配置，而是在考试概览中按单场考试配置。
