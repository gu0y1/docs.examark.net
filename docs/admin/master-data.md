# 学校基础数据

管理员设置用于维护学校运行 Examark 所需的基础数据。

## 学校信息

维护学校名称、简称和显示信息。学校信息会出现在页面顶栏和部分打印/报告场景中。

## 校历

校历包含学年和学期。新建考试时必须选择已有学期。

推荐维护顺序：

1. 创建学年。
2. 在学年下创建学期。
3. 调整排序。
4. 停用或删除未被引用的数据。

## 年级

年级用于班级和考试。年级可参与轮转，支持从七年级升到八年级等规则。

## 科目

科目用于考试创建和教师绑定。被考试引用的科目不应直接删除，通常转为停用。

## 班级

班级属于某个年级，是学生管理和考试考核班级的基础。
