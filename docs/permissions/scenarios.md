# 常见权限场景

本页用场景解释 Examark 的权限规则。核心原则是：能完成某个流程，不代表自动看到所有学生身份信息。

## 适用角色

- 学校管理员。
- 考试创建人。
- 扫描人员。
- 阅卷教师。
- 成绩查看人员。

## 你将完成什么

- 理解为什么不同角色看到的页面和字段不同。
- 判断问题是“没被加入考试”，还是“没有对应身份字段权限”。
- 知道遇到权限问题时应该找谁处理。

## 开始前需要

- 你已经知道自己在学校中的角色。
- 你知道自己是否被加入某场考试协作。

## 场景说明

| 场景 | 常见原因 | 应该怎么办 |
| --- | --- | --- |
| 我是阅卷老师，但看不到学生姓名 | 匿名阅卷默认隐藏身份字段 | 正常情况；只按题目答案和评分标准阅卷 |
| 我能上传扫描件，但看不到成绩 | 扫描权限和成绩权限分离 | 联系考试创建人确认是否需要成绩查看权限 |
| 我能编辑试卷，但看不到学生身份 | 试题编写不需要学生身份 | 正常情况；只需完成试题和答题卡 |
| 成绩发布了，但我还是看不到姓名 | 成绩查看和身份字段是两套权限 | 联系管理员确认是否授予身份字段权限 |
| scanner 能看原卷，grader 不能看 | scanner 需要绑定学生，grader 只需要评分 | 正常情况；避免阅卷过程被身份影响 |
| 我看不到管理员设置 | 当前账号没有任何设置相关权限 | 联系学校管理员确认角色绑定 |
| 管理员设置里缺少某个标签 | 只显示当前账号有权限的设置标签 | 联系管理员补充具体权限 |

## 典型排查步骤

1. 确认你是否登录了正确组织。
2. 确认你是否被加入本场考试协作。
3. 确认你在本场考试中的角色：创建人、考官、试题编写人、扫描人或阅卷人。
4. 确认你是否需要学生身份字段权限。
5. 如需调整，联系考试创建人或学校管理员。

## 完成后你应该知道

- 看不到学生姓名通常不是系统错误，而是匿名阅卷或字段裁剪。
- 扫描、阅卷、成绩查看、身份字段查看是不同能力。
- 管理员设置会按具体权限展示标签。

## 常见问题

**为什么我能看到成绩，但导出没有姓名和学号？**

导出也遵守同一套字段裁剪。没有学生身份字段权限时，系统不会在导出中泄露姓名和学号。

**为什么我在一场考试里能阅卷，在另一场考试里不能？**

考试内角色按单场考试配置。你需要被加入对应考试，并分配到对应题目。

**为什么权限变化后页面没有立即变化？**

先刷新页面或重新登录。如果仍不生效，请联系管理员确认角色绑定是否保存成功。
