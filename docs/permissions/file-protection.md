# 文件与导出保护

Examark 会按当前账号的角色和考试权限控制文件查看、打印、扫描原卷和成绩导出。页面上看不到的学生身份字段，也不会因为导出文件而绕过权限。

## 适用角色

- 考试创建人。
- 试题编写人。
- 扫描人员。
- 阅卷教师。
- 学校管理员。

## 你将完成什么

- 理解哪些文件需要权限才能查看或下载。
- 理解为什么导出文件也会裁剪姓名、学号和班级。
- 知道无权限时应该联系谁处理。

## 文件类型

| 文件或内容 | 通常谁需要查看 | 说明 |
| --- | --- | --- |
| 试卷 PDF | 考试创建人、考官、试题编写人 | 用于印刷和考试准备 |
| 答题卡 PDF | 考试创建人、考官、试题编写人 | 用于印刷和扫描识别 |
| 试题图片素材 | 试题编写人、阅卷相关人员 | 按考试任务需要展示 |
| 扫描原卷 | 扫描人员、考试创建人、授权人员 | 用于处理绑定、缺页和争议 |
| 题目裁剪图 | 被分配的阅卷教师 | 用于匿名阅卷 |
| 成绩导出 | 成绩查看或管理人员 | 按身份字段权限裁剪 |

## 导出保护

CSV、Excel 和 PDF 导出会执行同一套可见性规则。

| 当前账号能看到 | 导出中会包含 |
| --- | --- |
| 只有成绩 | 分数和匿名标识，不包含姓名、学号、班级 |
| 成绩 + 基础身份字段 | 可包含姓名、学号、班级等允许字段 |
| 扫描绑定信息 | 仅在扫描或授权排查场景中可见 |
| 敏感字段 | 只在确有授权时可见，普通成绩导出不建议包含 |

## 无权限时会怎样

无权限时，系统不会提供对应文件内容，也不会给出可间接访问文件的入口。

常见表现：

- 看不到下载按钮。
- 导出结果中缺少姓名、学号或班级。
- 阅卷页面只显示题目裁剪图，不显示整卷原图。
- 扫描原卷只对扫描处理或授权排查人员可见。

## 常见问题

**为什么我能看成绩，但导出没有姓名？**

成绩查看和身份字段是两套权限。你可以看分数，不代表可以导出姓名、学号或班级。

**为什么阅卷教师只能看到题目裁剪图？**

这是匿名阅卷设计。阅卷教师只需要看到自己负责题目的作答区域和评分标准。

**为什么扫描人员能看整卷原图？**

扫描人员需要处理学生绑定、缺页和异常卷，因此在扫描流程中可以看到必要原卷信息。

**需要临时查看更多字段怎么办？**

联系考试创建人或学校管理员说明原因。管理员应按最小必要原则开放权限，并在处理完成后收回不再需要的权限。
