# 答题卡

答题卡由系统根据试卷和模板生成，扫描识别依赖答题卡版式。

## 基础结构

答题卡包含：

- 考试标题。
- 条码。
- 页码。
- 学号填涂区。
- 作废标记。
- 客观题填涂区。
- 填空题答题区。
- 主观题答题区。

## 条码

条码用于识别答题卡、页码和模板版本。扫描时系统通过条码归组多页答题卡。

## 学号填涂

学号填涂支持配置：

- 前缀。
- 数字位数。
- 可填数字集合。

学号识别失败不会阻断整张答题卡扫描，系统允许后续人工绑定。

## 作废标记

如果作废区域被填涂，扫描页会被标记为作废。作废页不应进入正常阅卷流程。

## 增强扫描双色打印

增强扫描使用更淡的辅助色打印答题区域，降低扫描时将印刷内容误判为填涂的概率。
