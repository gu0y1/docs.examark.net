# 扫描识别

扫描识别负责把纸质答题卡转换为可复核、可阅卷的数据。

## 页面归一化

系统会对扫描图像做页面定位和归一化，使答题卡区域尽量对齐模板坐标。

## 客观题识别

客观题使用矩形框识别。系统会生成：

- 检测答案。
- 自动分数。
- 识别状态。
- 需要复核的异常标记。

## 主观题和填空题裁剪

系统会按模板坐标裁剪题目答题区域。阅卷人默认只能看到自己被分配题目的 `question_crop_image`，不能看到整卷 `full_scan_image`。

## 人工处理

当扫描件出现模糊、透视、阴影、缺页或版式不一致时，扫描人员可以结合识别结果进行人工处理。
