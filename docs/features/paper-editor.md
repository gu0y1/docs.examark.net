# 试题配置

试题配置页是 Examark 中最重要的编辑界面之一。

## 题组管理

题组用于组织试卷结构。教师可以新增、重命名、移动和删除题组。删除非空题组时，需要选择题目迁移目标。

## 题目编辑

每道题包含：

- 题型。
- 分值。
- 题干内容。
- 选项。
- 正确答案。
- 评分标准。
- 解析内容。
- 答题样式。

题号根据排序自动生成，不允许手动输入，以避免题号和答题卡布局不一致。

## 版本管理

保存试题会记录历史版本。进入预印后，当前版本被锁定。需要修改时，应先继续编辑并废弃当前预印或印刷版本。

## 打印信息

试卷信息配置可以控制：

- 学生信息栏。
- 考试时间。
- 考试地点。
- 考试时长。
- 考试须知。
- 得分表。
- 页脚页码。
