# 技术文档说明

本栏目面向 IT、运维和开发人员，不是普通教师的日常操作指南。

普通教师、扫描人员和阅卷教师遇到考试流程问题时，请优先查看：

- [普通用户 FAQ](../faq/index.md)
- [故障处理](../faq/troubleshooting.md)
- [扫描与识别 SOP](../scanning-printing/sop.md)

## 适用角色

- IT / 运维人员。
- 平台管理员。
- 负责部署、域名、证书、备份和服务器排错的人员。

## 内容边界

技术与部署栏目可以包含：

- Linux 部署。
- 域名和组织入口。
- 截图补充清单和文档站发布。
- Nginx、证书和服务状态检查。
- 字体和 PDF 渲染依赖。
- 数据备份和恢复建议。
- 开发环境排错。

技术内容不应混入普通用户 FAQ，避免学校老师误以为需要处理服务器或开发命令。

## 当前部署约定

- Examark 当前按 Linux-only 部署维护。
- Windows、WSL、PowerShell 和本地前端开发服务器问题不属于普通学校用户手册。
- 如果技术排错中提到这些词，应明确标注为 IT / 运维 / 开发排错内容。

## 常见任务

- 检查服务是否运行。
- 检查域名和 HTTPS 是否可用。
- 检查中文字体和 Chromium 是否满足 PDF 渲染要求。
- 检查数据库和租户 storage 备份。
